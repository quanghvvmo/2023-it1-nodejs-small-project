# SMALL PROJECT

## Intro

Small project

## Techs
Languages: Javascript
Framework: ExpressJs
Database: MySQL (Sequelize)

## To prepare project

Create .env file in root directory with the following format: 
```
HTTP_PORT=
DB_DATABASE=
DB_USERNAME=
DB_PASSWORD=
DB_HOST=
DB_PORT=
DB_DIALECT=
TOKEN_SECRET=
TOKEN_EXPIRY=
```

## To run project

```shell
> npm install
> npm start
```
